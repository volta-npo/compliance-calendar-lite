# Compliance Calendar Lite v1.0 Product Specification

## Workbench

Compliance Calendar

## Domain table

Recurring deadlines

## Metrics

- Deadline Coverage
- Overdue Risk
- Evidence Discipline

## Domain rows

- Deadline type entered
- Jurisdiction noted
- Owner assigned
- Frequency selected
- Next due date generated
- Evidence requirement set
- Completion log started
- ICS calendar exported

## Generated deliverables

- ICS compliance calendar
- Owner task CSV
- Evidence log

## Validation gates

- Every deadline needs owner/evidence type
- Completed items need date
- Legal disclaimer included
